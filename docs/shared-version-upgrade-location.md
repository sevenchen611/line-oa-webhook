# Shared Version Upgrade Location

For every completed reusable improvement, read this shared playbook:

```text
D:\Codex_project\AM_SHARED_VERSION_UPGRADES\VERSION_UPGRADE_PLAYBOOK.md
```

That file defines how to create an improvement version, update the shared registry, and update this project's `docs/project-improvement-manifest.md`.

This project must keep its own LINE, Notion, Render, and environment values. Do not copy values from HOZO AM.
