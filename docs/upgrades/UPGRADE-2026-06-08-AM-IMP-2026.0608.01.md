# Upgrade Version Record

Version: `AM-IMP-2026.0608.01`

Improvement name: Project data isolation guard

Target project: 7AM

Upgrade date: 2026-06-08

Owner: Seven / Codex

## Purpose

Prevent SevenAM from accidentally reading or writing another Assistant Manager project's Notion data sources.

The guard checks Notion data source access before API calls that query a data source or create a page under a data source.

| Item | Included? | Notes |

## Data Isolation Check

| Check | Result |
| --- | --- |
| Uses only SevenAM env names | Pass. Uses `SEVEN_*` variables only. |
| Copies HOZO Notion IDs | Pass. No HOZO IDs copied. |
| Copies HOZO token or Render service ID | Pass. No secrets or service IDs copied. |
| Blocks suspicious Notion data source titles | Pass. Only SevenAM-like titles such as Seven, Codex, LINE, 總控, 任務, 專案, 會議, 每日, 風險, 決策, 權責 are allowed. |
