## LINE Commands

| Command | Behavior |
| --- | --- |
| `Seven Junior，我們開始做任務校準` | Starts or resumes calibration and sends the next item when no older item is waiting. |
| `Seven Junior，任務校準暫停` | Pauses calibration so normal replies do not trigger another item. |
| `Seven Junior，任務校準狀態` | Reports calibrated count, remaining count, and pending reply count. |

Each outgoing review item shows progress as:

```text
【判斷校準】2/80
已校準：1｜尚未校準：79
```

## Next Operating Steps

1. The controller sends `Seven Junior，我們開始做任務校準` in LINE when available.
2. The controller replies in LINE with direction, reason, and optional reusable rule.
3. The webhook records the reply, updates the source task, extracts a rule when possible, and sends the next item while active.
4. The controller sends `Seven Junior，任務校準暫停` when unavailable.
